Features

Torch is more than just a web host. It can be your development playground! We've got a bunch of features to assist you.

See the full list of features

Features:

Login with your FSUID

Never juggle passwords again! SSH and SFTP access use your FSUID and FSU password.

MySQL Database Access

Harness the power of MySQL to create a web application or explore Structured Query Language.

Git

We encourage students to use Git to manage version control of their projects.

Read more information about Git.

TL;DR

Forget what a Linux command does? tldr has you covered.

Simply type tldr $command and you'll get a cheat sheet for that command, with examples of how to use it!

Read more information about tldr.

Packages Installed

Just some of the packages available on Torch:

  • zsh
  • python3
  • npm
  • git
  • tldr
  • vim

Privacy

Your home folder has strict permissions, so that only you can view its contents. Keep your million-dollar idea safe.

Just remember, anything placed inside your public_html folder will be viewable online by anyone in the world!
