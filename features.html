Features

Torch is loaded with features to assist development.

Are we missing something awesome? Contact the CCI HelpDesk.

Features of Torch:

Login with your FSUID

Never juggle passwords again.

Login with your FSUID and password, and your MySQL information will be emailed to your @my.fsu.edu address.

Git

Git is installed and encouraged on torch.

Git clone your favorite projects, or anything interesting. Even install projects with multiple databases, using FSUID_ for multiple tables.

Example:

web15c@torch:~$ git clone https://github.com/cci-help/torch.git

TL;DR

Forget what a Linux command does? tldr has you covered.

Simply type "tldr command" and you'll get a cheatsheet for common Linux commands, including examples.

Example:

web15c@torch:~$ tldr tar

- Create an archive from files:

tar cf target.tar file1 file2 file3

Packages Installed

Here's some common packages that are installed on torch:

  • zsh
  • python3
  • npm
  • git
  • tldr
  • vim

Privacy

Your home folder has strict permissions ('0700'). Keep your million-dollar idea safe.

Example:

drwx------ 4 web15c domain users 4096
