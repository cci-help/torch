Faculty Information

Torch is designed to make instruction easy. We built the sandbox, so now your students can build the sandcastle!

Get information for your class

Here's how Torch can work for you:

Web Hosting for Students

If you're teaching a class on HTML, CSS, PHP, etc. your students will need a place to host their projects. Torch makes hosting student sites super simple and secure. Each student has a home folder named after their FSUID. Inside is a web directory for uploading their site files. You can easily see your students' work in the browser.

Databases for Students

Are you teaching a course on databases? Your students could benefit from a MySQL server for in-class exercises and personal exploration. Torch is a MySQL server and each student is given their own MySQL user and database for learning. Let Torch supplement your course by providing hands-on experience with database administration.

Automatic Student Enrollment

Any student enrolled in a CCI course will have access to Torch on day one of class. Students will be able to SSH or SFTP with their FSU credentials, and MySQL information will be emailed to them automatically.

Please note that teachers will need to contact the CCI HelpDesk to get instructor access to Torch.

Security

Students no longer share the same passwords. Each student has unique SSH/SFTP and MySQL information.
Torch also uses a valid SSL certificate for all student websites - you can now create real, secure login pages. As an instructor, you can also securely browse your students' web files and databases for simplified grading.

Need help?
